# Synonyms — LEVEL 3A (Padārtha Atoms)

Synonym atoms provide **word-by-word meanings**
for verses that traditionally include padārtha.

They are not translations.
They are not glosses.
They are structural aids.

Rules:
- Verse-specific
- One synonym set per file
- No interpretation
- No prose explanation

Synonyms support reading — not doctrine.
